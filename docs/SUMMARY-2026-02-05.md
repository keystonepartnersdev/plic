## 오늘 해결한 문제 (2026-02-05)

### 1. FAQ 안 보이는 문제
- **증상**: "등록된 FAQ가 없습니다" 표시
- **원인**: 카테고리가 영문(`service`)인데 백엔드는 한글(`서비스 이용`) 반환
- **해결**: 모든 카테고리를 한글로 통일

### 2. React Error #310
- **증상**: 거래 상세 페이지 무한 로딩
- **원인**: `useCallback` 훅의 의존성 문제로 무한 재렌더링
- **해결**: `useCallback` → 일반 함수로 변경

## 주요 기능

```
[회원가입] → [로그인] → [거래 생성] → [결제] → [송금]
    │           │           │           │         │
카카오 연동  JWT 인증    금액/수취인   PG 결제   운영팀 처리
                          서류 첨부
```

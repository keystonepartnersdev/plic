## 왜 PLIC인가요?

PLIC은 카드 결제의 편리함과 계좌 송금의 범용성을 결합한 새로운 금융 서비스입니다. 급한 송금도 부담 없이 가능합니다.

### 6가지 핵심 장점

1. **결제일까지 여유**
   현금 여유가 없을 때, 신용카드로 결제가 가능하기에 여유로운 자금흐름을 만들 수 있습니다.

2. **원금 전액 송금 보장**
   수취인은 송금 원금을 100% 그대로 받습니다. 100만원을 보내면, 100만원이 도착합니다.

3. **D+3일 이내 송금**
   결제 완료 즉시 운영팀 검토 후 D+3(영업일 기준)일 이내에 송금됩니다.

4. **어떤 계좌든 OK**
   국내 모든 은행 계좌로 송금할 수 있습니다. 수취인이 PLIC 회원이 아니어도 괜찮습니다.

5. **카드 혜택 그대로**
   카드 포인트, 할인, 할부 등 기존 카드 혜택을 그대로 누릴 수 있습니다.

6. **실시간 거래 내역**
   모든 거래 상태를 실시간으로 확인할 수 있습니다. 투명한 거래 이력 관리가 가능합니다.

## 이용 방법: 간단하게 송금하세요

복잡한 절차 없이, 카드 하나로 누구에게나 송금할 수 있습니다.

### 4단계 프로세스

1. **회원가입**
   간편하게 사업자 정보로 가입하세요. 1분이면 충분합니다.

2. **거래 등록**
   거래 유형, 금액, 수취인 정보, 증빙을 등록합니다.

3. **카드결제**
   본인 명의의 카드로 결제해주세요.

4. **송금완료**
   결제가 완료되면 운영팀 검수 후 수취인에게 원금이 즉시 송금됩니다.

## 보안: Security First

표준 보안 체계로 안전하게 지켜드립니다

PLIC은 보안 표준을 준수하며, 고객님의 소중한 정보를 철저하게 보호합니다.

### 4가지 보안 기능

1. **카드 정보 비접촉**
   카드 번호는 PLIC 서버를 거치지 않습니다. PG사 SDK를 통해 토큰화되어 처리됩니다.

2. **KMS 암호화**
   민감 정보는 AWS KMS AES-256으로 암호화하여 저장합니다.

3. **본인인증 필수**
   PASS 본인인증과 3D Secure로 부정 사용을 방지합니다.

4. **이상거래 탐지 (FDS)**
   실시간 이상거래 탐지 시스템으로 의심 거래를 사전에 차단합니다.

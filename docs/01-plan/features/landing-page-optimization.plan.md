### [2] Hero — 태그 교체 (색상 + 내용)

**유지 항목:**
- "사업자 전용 서비스" 상단 배지 유지
- 롤링 텍스트 유지 (월세를 / 거래대금을 / 계약금을)
- 메인 헤드라인 유지 (`xx를 카드로 결제하다. PLIC`)
- 서브카피 유지 ("월세, 거래대금 등 현금 거래 사업비 카드로 결제하시면 송금됩니다.")

**변경 항목 — 하단 태그 교체:**
- AS-IS: 회색 태그 5개 (`거래대금` / `사업장월세` / `인건비` / `자재비` / `장비 대여료`)
- TO-BE: **주황색 계열** 태그 1개
  ```
  업계 최저 수수료!  월세 2.9%,  최초 거래 1.8% EVENT 진행 중!
  ```
- 색상: `bg-orange-100 text-orange-700` (또는 `bg-amber-100 text-amber-700`) 계열

> **섹션 타이틀**: "왜 PLIC인가요?" 유지  
> **카드 수**: 6개 유지, 내용만 전면 교체

| # | 아이콘 | 카드 제목 | 카드 설명 |
|---|--------|-----------|-----------|
| 1 | CreditCard | 무이자 할부 가능! | 카드사에서 제공하는 무이자 할부 기간동안 이용 가능하시며 총 12개월까지 할부 결제가 가능합니다. |
| 2 | Tag (또는 Percent) | 업계 최저 수수료! | 월세는 상시 2.9% 수수료 적용! 첫 거래 시, 수수료 1.8% 적용 쿠폰도 지급합니다!(EVENT 사항) |
| 3 | Gift | 카드 혜택도 그대로 적용! | 카드 포인트 적립, 할인, 항공마일리지 적립 등 기존 카드 혜택을 그대로 누릴 수 있습니다. |
| 4 | BarChart3 | 대출보다 압도적으로 유리한 PLIC | 신용대출 평균 이자율 6~13%, 카드론 12~17%. PLIC 수수료는 월 2.9~3.3%. 신용점수 하락도 전혀 없습니다. |

```
┌────────────────────────────────────────────────────┐
│           업계 최저 수수료!                         │
│   월세 2.9%  ·  최초 거래 1.8% EVENT 진행 중!     │  ← 후킹 헤드라인 (큰 폰트, 강조)

│  부연 설명 텍스트 (중간 폰트)                      │

│  실질 수수료 부담은 생각보다 훨씬 저렴합니다.      │
├────────────────────────────────────────────────────┤
│ ┌──────────┐  ┌──────────┐  ┌──────────┐          │
│ │  1.8%   │  │  2.9%   │  │  3.3%   │          │
```

### [8] TrustSection — 타이틀·설명 폰트 크기 조정

**목표:** Security 섹션의 타이틀·설명 텍스트 크기와 동일하게 맞춤

**Security 섹션 기준 (현재 코드):**
- 섹션 타이틀: `text-4xl md:text-5xl font-bold`
- 섹션 설명: `text-lg text-gray-500`
- 카드 제목: `text-xl font-bold`
- 카드 설명: `text-sm md:text-base`

**TrustSection 현재 vs 목표:**
| 요소 | 현재 | 목표 (Security 기준) |
|------|------|---------------------|
| 섹션 타이틀 | `text-4xl md:text-5xl font-black` | `text-4xl md:text-5xl font-bold` |
| 섹션 설명 | `text-lg text-gray-600` | `text-lg text-gray-500` |
| 카드 제목 | `font-bold` (크기 없음) | `text-xl font-bold` (명시) |
| 카드 설명 | `text-sm` | `text-sm` (유지) |

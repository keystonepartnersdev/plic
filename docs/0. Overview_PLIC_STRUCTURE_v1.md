### 2.1 사용자 관련 (`src/types/user.ts`)

#### 타입 정의

| 타입명 | 값 | 용도 |
|-------|-----|------|
| `TUserStatus` | `'active'` \| `'suspended'` \| `'pending'` \| `'withdrawn'` | 사용자 계정 상태 |
| `TUserGrade` | `'basic'` \| `'platinum'` \| `'b2b'` \| `'employee'` | 사용자 등급 |
| `TSocialProvider` | `'kakao'` \| `'naver'` \| `'google'` \| `'apple'` \| `null` | 소셜 로그인 제공자 |
| `THistoryActor` | `'member'` \| `'admin'` \| `'system'` | 히스토리 변경 주체 |
| `THistoryField` | `'signup'` \| `'status'` \| `'grade'` \| ... | 히스토리 변경 항목 |

#### IUser 인터페이스

```typescript
interface IUser {
  uid: string;                         // 사용자 고유 ID (U + YYMMDD + 랜덤6자)
  name: string;                        // 이름
  phone: string;                       // 전화번호
  email?: string;                      // 이메일 (선택)

  // 인증 관련
  authType: 'direct' | 'social';       // 인증 방식
  socialProvider: TSocialProvider;     // 소셜 로그인 제공자
  socialId?: string;                   // 소셜 ID
  isVerified: boolean;                 // 본인인증 여부
  verifiedAt?: string;                 // 인증 일시

  // 상태 및 등급
  status: TUserStatus;                 // 계정 상태
  grade: TUserGrade;                   // 회원 등급
  feeRate: number;                     // 수수료율 (%)
  isGradeManual: boolean;              // 수동 등급 부여 여부

  // 한도
  monthlyLimit: number;                // 월 한도
  usedAmount: number;                  // 월 사용량

  // 동의 항목
  agreements: {
    service: boolean;                  // 서비스 이용약관
    privacy: boolean;                  // 개인정보 처리방침
    thirdParty: boolean;               // 제3자 정보제공 동의
    marketing: boolean;                // 마케팅 수신 동의
  };

  // 누적 정보
  totalPaymentAmount: number;          // 총 결제액
  totalDealCount: number;              // 총 거래 건수
  lastMonthPaymentAmount: number;      // 전월 결제액 (자동 등급 산정용)

  // 히스토리
  history: IUserHistory[];             // 변경 이력

  // 일시 정보
  createdAt: string;                   // 가입일
  updatedAt: string;                   // 수정일
  lastLoginAt?: string;                // 마지막 로그인
}
```

#### IDeal 인터페이스

```typescript
interface IDeal {
  did: string;                         // 거래 ID (D + YYMMDD + 랜덤6자)
  uid: string;                         // 사용자 ID

  // 거래 기본 정보
  dealName: string;                    // 거래명
  dealType: TDealType;                 // 거래 종류
  status: TDealStatus;                 // 거래 상태
  revisionType?: 'documents' | 'recipient';  // 보완 요청 유형
  revisionMemo?: string;               // 보완 요청 메모

  // 금액 정보
  amount: number;                      // 송금액
  feeRate: number;                     // 수수료율
  feeAmount: number;                   // 수수료액
  totalAmount: number;                 // 총액 (송금액 + 수수료)

  // 할인 정보
  discountCode?: string;               // 할인코드
  discountAmount: number;              // 할인금액
  finalAmount: number;                 // 최종결제액

  // 상대방 정보
  recipient: IRecipientAccount;        // 수취인 계좌
  senderName: string;                  // 보내는 분

  // 첨부 서류
  attachments: string[];               // 첨부 파일 URL 배열

  // 결제 정보
  paymentId?: string;                  // 결제 ID
  isPaid: boolean;                     // 결제 완료 여부
  paidAt?: string;                     // 결제 일시

  // 송금 정보
  isTransferred: boolean;              // 송금 완료 여부
  transferredAt?: string;              // 송금 일시

  // 이력
  history: IDealHistory[];             // 거래 이력

  // 일시 정보
  createdAt: string;
  updatedAt: string;
}
```

### 3.2 useUserStore 상세

**상태 (State)**
| 필드명 | 타입 | 설명 |
|-------|------|------|
| `currentUser` | `IUser \| null` | 현재 로그인 사용자 |
| `isLoggedIn` | `boolean` | 로그인 여부 |
| `registeredCards` | `IRegisteredCard[]` | 등록된 카드 목록 |
| `users` | `IUser[]` | 전체 회원 목록 |

**액션 (Actions)**
| 함수명 | 역할 |

| Anti-pattern | 설명 | 수정 방법 |
|-------------|------|----------|
| `waitForURL().catch(() => {})` | 실패해도 무시 → 테스트 무의미 | `expect(page).toHaveURL()` 변환 |
| `expect(x \|\| true).toBeTruthy()` | 항상 true → 테스트 무의미 | 직접 `expect(element).toBeVisible()` 변환 |
| `toBeGreaterThanOrEqual(0)` | 0 이상 (항상 true) → 테스트 무의미 | `toBeGreaterThanOrEqual(1)` 또는 `toBeVisible()` 변환 |
| `if (visible) { click; }` | 요소 없으면 skip → assertion 없음 | `expect().toBeVisible()` 후 클릭 + 결과 assertion |
